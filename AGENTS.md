- 本仓库当前默认使用中文沟通、中文注释、英文日志。
- 当前实现采用单镜像部署优先原则，默认以本地 JSON 持久化和本地文件存储完成 MVP。
- UI 需要优先贴合 `ui/ui.pen` 的深色极简风格，避免引入多余视觉元素。
- 每次修改后需要同步说明：问题原因、修复思路、实现差异。
- 阅读器交互默认禁止平滑动画；进入书籍、目录跳转、划线跳转都应直接定位到上次或目标阅读位置。
- 阅读器目录栏与右侧划线栏需要支持拖拽调宽，并记忆用户宽度偏好。
- 新增高亮后，正文原文中的对应片段也要立即呈现高亮效果；不展示“高亮已添加”提示。
- EPUB/正文内容抽取时必须保留块级结构、段落换行，并解码常见 HTML 实体，避免正文被压成一整段。
- HTML/EPUB 内容解析优先使用结构化解析器（如 parse5）与实体解码库，而不是正则剥标签；高亮定位优先保留章节级 locator（如 chapterHref）再做文本兜底。
