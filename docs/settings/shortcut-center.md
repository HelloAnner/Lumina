# 快捷键中心

> 对应模块：设置 / 阅读设置
> 状态：已实现

---

## 目标

统一管理两类文本交互快捷键：

- 阅读器划词操作
- 知识库笔记编辑器

用户可以在设置页修改单键或组合键，保存前系统会校验：

- 应用内重复绑定
- 浏览器/系统保留快捷键

命中冲突时，不允许保存。

## 数据结构

快捷键配置挂在 `ReaderSettings.keyboardShortcuts` 下，分为两个分组：

- `reader`
- `noteEditor`

其中 `reader.highlightShortcuts` 继续保留为兼容字段，并与 `keyboardShortcuts.reader` 同步。

## 校验规则

保存前与服务端都会执行统一校验：

- 空绑定：不允许
- 重复绑定：不允许
- 系统保留快捷键：不允许

典型保留快捷键包括：

- `Cmd+W`
- `Cmd+T`
- `Cmd+Space`
- `Ctrl+W`
- `Ctrl+T`
- `Alt+Tab`

## 生效范围

### 阅读器

- 黄色/绿色/蓝色/粉色高亮
- 打开笔记

### 笔记编辑器

- 批注
- 加粗 / 斜体 / 高亮 / 删除线 / 行内代码 / 链接
- 复制块
- 块上移 / 下移
- 标题 1 / 2 / 3
- 段落
