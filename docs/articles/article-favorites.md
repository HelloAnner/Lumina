# 文章收藏

## 目标

- 用户可在文章列表与阅读页收藏文章
- 收藏文章永远不被自动清理或永久删除
- 收藏后系统基于文章标题与已有划线，自动生成并补全主题标签

## 交互

### 文章列表

- 顶部筛选新增 `收藏`
- 每行文章右侧新增星标按钮
- 已收藏文章显示常亮星标与轻量收藏标记

### 文章阅读页

- 顶栏右上角新增收藏按钮
- 点击后立即切换收藏状态
- 收藏成功后异步触发 AI 自动打标签

## 数据行为

- `ScoutArticle` 新增：
  - `favorite?: boolean`
  - `favoritedAt?: string`
- 自动归档清理时跳过收藏文章
- 手动永久删除收藏文章时直接拒绝

## 自动标签

- 输入：文章标题 + 已有划线文本 + 批注
- 输出：1~3 个高质量主题标签
- 优先匹配已有 `ArticleTopic`
- 无匹配则自动创建新主题，再绑定到文章

## API

- `POST /api/articles/:id/favorite`
- `POST /api/articles/:id/unfavorite`

## 视觉

- 星标采用克制的暖色强调，不做高饱和闪烁
- 收藏筛选沿用现有顶部 tab
- 收藏文章在列表中只做轻量强调，不破坏整体宁静感
