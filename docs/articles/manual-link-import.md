# 手动添加文章链接

## 目标

在文章列表页支持用户手动粘贴文章 URL，并通过回车或按钮触发解析，把文章直接加入文章库。

## 交互

- 在文章列表头部新增 `添加链接` 按钮，位于搜索框右侧
- 点击后打开居中的轻量弹窗，输入框自动聚焦
- 用户粘贴 URL 后按回车，立即开始解析并入库
- 解析中在弹窗内展示 loading 状态与当前文案，避免用户不确定系统是否正在工作
- 成功后关闭弹窗，顶部 toast 提示 `文章已添加`，并把新文章插入列表顶部
- 如果 URL 已存在，不重复创建，关闭弹窗并提示 `文章已在列表中`
- 解析失败时保留弹窗与输入内容，给出温和错误提示，并允许重新尝试

## 数据行为

- 新增 `POST /api/articles/import` 接口，接收 `{ url }`
- 服务端先对 URL 做归一化，避免因追踪参数导致重复文章
- 使用现有 Readability 提取正文
- 成功后创建一篇 `status=ready` 的文章，来源标记为手动添加
- 若正文提取失败，返回明确错误，不创建空文章

## 视觉

- 交互沿用现有温和深色面板语言
- 按钮为 shadcn 原生按钮，尺寸与搜索框高度对齐
- 弹窗保持大圆角、1px 边框、低对比背景，不引入夸张动效
- 输入框下方补充一行辅助文案：支持粘贴单篇文章链接，回车后开始解析
